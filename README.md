# the-extended-Nei-Gojobori-method
This program is designed for the incorporating mutation spectrum into the Nei-Gojobori method (Nei and Gojobori, 1986) by using the relative rates of six mutation types (AT -> TA, AT -> CG, AT -> GC, CG -> TA, CG -> AT, CG -> GC) to calculate S and N for a sequence. This program is written in Python and requires Python >= 3.10.
